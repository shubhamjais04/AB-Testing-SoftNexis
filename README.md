# 🧪 A/B Testing & Hypothesis Testing

A complete A/B testing and statistical hypothesis testing workflow on a large-scale e-commerce dataset, determining whether a new webpage design significantly improves user conversion rates.

---

## 📌 Project Overview

A/B testing is one of the most powerful tools in a data scientist's toolkit for making data-driven business decisions. This project demonstrates a full hypothesis testing pipeline — from formulating the null hypothesis to running statistical tests and drawing actionable conclusions on real e-commerce data.

---

## ✨ What's Covered

- 📥 Data Loading & Cleaning — Handling 294,000+ user interaction records
- 📝 Hypothesis Formulation — Null and alternative hypothesis definition
- 📊 Exploratory Analysis — Conversion rate comparison between control and treatment groups
- 🔬 Two-Proportion Z-Test — Testing statistical significance of conversion difference
- 📐 Confidence Interval Visualization — Visual representation of uncertainty
- χ² Chi-Square Test — Independence testing between group and conversion
- 📏 Independent T-Test — Mean comparison between groups
- ⚡ Power Analysis — Determining minimum sample size for reliable results
- 📋 Final Summary — Business recommendation based on statistical findings

---

## 📊 Dataset

- **Records:** ~294,000 user interactions
- **Domain:** E-commerce user behavior
- **Columns:** `user_id`, `timestamp`, `group`, `landing_page`, `converted`
- **Groups:** Control (old page) vs Treatment (new page)
